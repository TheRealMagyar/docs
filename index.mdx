## Welcome to the ShadowCorp SDK

The **ShadowCorp SDK** is the official client library for interacting with the ShadowCorp NEAR Blockchain API. It provides a clean, typed, and intuitive interface to access on-chain data, analytics, and forensic tools â€“ all with full TypeScript support.

Whether you're building a block explorer, wallet analytics dashboard, DeFi tracker, or conducting blockchain investigations, this SDK makes it easy to fetch blocks, transactions, account details, token/NFT transfers, validator info, anomaly alerts, RPC calls, and read-only graph queries.
