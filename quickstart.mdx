## Get started in three steps

Install the SDK, authenticate, and make your first API call.

### Step 1: Installation

The ShadowCorp SDK is available on npm and works in both JavaScript and TypeScript projects.

<Tip>Requires Node.js 18 or higher</Tip>

<CodeGroup>
```bash npm
npm install shadowcorp
```
```bash yarn
yarn add shadowcorp
```
```bash pnpm
pnpm add shadowcorp
```

</CodeGroup>

### Step 2: Authentication

All requests (except /health and /) require an API key. Get your free API key at https://dashboard.shadowcorp.dev

<AccordionGroup>

  <Accordion title="Environment variable (recommended)">
    Store your API key securely in an `.env` file:
```env
SHADOWCORP_API_KEY=your_api_key_here
```

The SDK will automatically read it from `process.env.SHADOWCORP_API_KEY`.

  </Accordion>

  <Accordion title="Direct instantiation">
    Pass the key directly when creating the client:
```ts
const sdk = new ShadowCorpSDK('your_api_key_here');
```

  </Accordion>

</AccordionGroup>

### Step 3: Make your first request

<CodeGroup>
```ts TypeScript
import { ShadowCorpSDK } from 'shadowcorp';

const sdk = new ShadowCorpSDK(process.env.SHADOWCORP_API_KEY || 'your_api_key_here');

// Get network statistics
const stats = await sdk.stats.getNetworkStats();
console.log('Current block height:', stats.latest_block_height);
console.log('Total transactions:', stats.total_transactions);

// Get full account data
const account = await sdk.accounts.getFull('therealmagyar.near');
console.log("Account balance:", BigInt(account.state.amount));
console.log('Tokens owned:', account.tokens?.length ?? 0);

````
```js JavaScript
const { ShadowCorpSDK } = require('shadowcorp');

const sdk = new ShadowCorpSDK(process.env.SHADOWCORP_API_KEY || 'your_api_key_here');

// Get network statistics
const stats = await sdk.stats.getNetworkStats();
console.log('Current block height:', stats.latest_block_height);

// Get a specific block
const block = await sdk.blocks.getByHeight(178978873);
console.log('Block hash:', block.hash);
````

</CodeGroup>

<Note>
  All methods return plain JSON objects. Errors are thrown as custom classes
  (`AuthError`, `RateLimitError`, etc.) for easy handling.
</Note>
